\### 4. 仮説出力の指定テンプレート（Hypothesis-output template）



今後のステップで私が仮説を出力する際は、バックテスト（BT）への組み込みやすさを最優先とし、以下のフォーマットに厳格に従って出力します。

```markdown
\*\*推奨買い目 (Ticket Pattern):\*\* \[例: 1 - 2,3 - 2,3,4]

\- `races.wind\_speed\_m` >= 5

\- `races.wind\_direction\_code` == '向かい風' (※辞書定義に準拠)

\- `entries\[lane=1].exhibition\_time\_rank` >= 3

\- `entries\[lane=2].st\_diff\_from\_inside` <= -0.10



\*\*期待されるメカニズム (Expected Mechanism):\*\*

\[例: 強い向かい風で1コースの出足が鈍る中、1コースの展示タイムが悪く、かつ2コースが1コースよりもスタート展示で先行している場合、2コースの直まくり、または差しが決まりやすくなる構造的な歪みが発生する。]



\*\*最小サンプル数のガードレール (Minimum Sample Guardrail):\*\*

\[例: 該当条件の発生が期間内で N=50 以上であること。]



\*\*一般化の根拠 (Why it might generalize):\*\*

\[例: 特定の選手名やモーター番号に依存せず、流体力学的な「風」と「スタート隊形」の物理的な相対関係に基づいているため、他期間でも再現する可能性が高い。]

```



\### 5. 情報漏洩（Information Leakage）と過学習（Overfitting）への警告



バックテストに向けて、以下の点に注意してデータを運用してください。



1\.  \*\*未来情報の漏洩 (Look-ahead bias):\*\* `entries` テーブルに「最終オッズ」や「レース後の確定ST」が含まれないようにしてください。必ず「締め切り前に取得可能なデータ（展示情報など）」だけで条件式を組む必要があります。（現在の辞書を見る限り、オッズは除外されており安全な設計です）。

2\.  \*\*サンプルサイズの罠:\*\* 4ヶ月間という期間は、特定の競艇場（例: 冬季の荒れる鳴門など）の特定の季節要因を過大評価するリスクがあります。そのため、AIが「場コード」と「極端な風速」をピンポイントで組み合わせた複雑な条件式を出してきた場合は、過学習の可能性が高いため注意が必要です。

3\.  \*\*欠損値の扱い:\*\* 「展示不参加」や「欠場」があった場合の `exhibition\_time` などの扱い（0にするのかnullにするのか）がBT時にエラーの原因になりやすいため、AIに出力させる条件式には `IS NOT NULL` などのセーフティネットを組み込むことを推奨します。

\### Hypothesis ID: \[H-001]



\*\*Target Ticket Type:\*\* 2連単 (Exacta)



\*\*Ticket Pattern:\*\* 2-1, 2-3, 2-4



\*\*Condition Block:\*\*

\- `entries\[lane=1].exhibition\_time\_rank` >= 4

\- `entries\[lane=1].exhibition\_time\_rank` IS NOT NULL

\- `entries\[lane=2].exhibition\_time\_rank` <= 2

\- `entries\[lane=2].st\_diff\_from\_inside` <= -0.05

\- `entries\[lane=2].st\_diff\_from\_inside` IS NOT NULL



\*\*Expected Mechanism:\*\*

1コースのモーター（展示タイム）が劣勢であり、かつ2コースのモーターが上位である状況。さらにスタート展示において、2コースが1コースよりも0.05秒以上先行（踏み込んでいる）している場合、1コースは第1ターンマークで物理的に抵抗できず、2コースの直差しまたは同体まくりが決まる確率が構造的に高まる。



\*\*Minimum Sample Guardrail:\*\*

N >= 80



\*\*Why It Might Generalize:\*\*

選手の知名度や勝率といった長期的指標ではなく、「当日のモーター推力」と「スタートの踏み込み」という純粋な物理的優位性に基づいているため、期間や場を問わず再現しやすい。



\*\*Complexity Check:\*\*

Low

\*\*Ticket Pattern:\*\* 4-1, 4-5, 4-6



\*\*Condition Block:\*\*

N >= 60



\*\*Why It Might Generalize:\*\*

\### Hypothesis ID: \[H-003]



\*\*Target Ticket Type:\*\* 3連単 (Trifecta)



\*\*Ticket Pattern:\*\* 1-2-3, 1-3-2, 1-2-4



\*\*Condition Block:\*\*

\- `races.a1\_count` == 2

\- `entries\[lane=1].racer\_class` == 'A1'

\- `entries\[lane=2].racer\_class` == 'A1' OR `entries\[lane=3].racer\_class` == 'A1'

\- `entries\[lane=4].exhibition\_time\_rank` >= 3

\- `entries\[lane=4].exhibition\_time\_rank` IS NOT NULL



\*\*Expected Mechanism:\*\*

レース全体にA1選手が2名のみ存在し、それが1コースと「2コースまたは3コース」に配置されている番組構成。この2名で強固な「インの壁」が形成されるため、外側からのまくりやまくり差しが物理的に届かなくなる。さらに4コースの展示タイムが平凡（3位以下）であれば、ダッシュからの強襲リスクも低下し、内枠中心の堅い決着が強制される。



\*\*Minimum Sample Guardrail:\*\*

N >= 100



\*\*Why It Might Generalize:\*\*

「強い選手が内側に固まることで外側の勝機を物理的にブロックする」という、番組構成そのものの構造的な特性を利用しているため、特定の季節や水面に依存しない。



\*\*Complexity Check:\*\*

Medium

\### Hypothesis ID: \[H-005]



\*\*Target Ticket Type:\*\* 3連単 (Trifecta)



\*\*Ticket Pattern:\*\* 5-1-2, 5-1-6



\*\*Condition Block:\*\*

\- `entries\[lane=5].win\_rate\_rank` >= 4

\- `entries\[lane=5].win\_rate\_rank` IS NOT NULL

\- `entries\[lane=5].exhibition\_time\_rank` == 1

\- `entries\[lane=4].st\_diff\_from\_inside` >= 0.05

\- `entries\[lane=4].st\_diff\_from\_inside` IS NOT NULL



\*\*Expected Mechanism:\*\*

5コースの選手が全国勝率では下位（4位以下）であるため一般的に軽視されやすいが、当日の展示タイムでは単独1位を叩き出している「隠れ超抜モーター」状態。さらに、隣の4コースがスタート展示で内側よりも遅れている（st\_diff\_from\_inside >= 0.05）ため、5コースから一気にまくり差しを狙える進路が開いている。1コースは展開のあおりを受けても残しやすい。



\*\*Minimum Sample Guardrail:\*\*

N >= 40

「長期勝率（過去の能力）」と「当日の展示タイム（現在の機力）」のギャップが存在する時、オッズ（期待値）の歪みが最も大きくなる。このギャップに「隣の艇の遅れ」という展開の利が重なるため。

High

\- \*\*最も堅牢（Robust）と思われる仮説:\*\* `\[H-001]` および `\[H-003]` です。`\[H-001]` は純粋なモーター推力とスタート位置の物理的な優位性に基づいており、ノイズが入りにくいです。`\[H-003]` は「級別による壁の形成」という番組マンの意図と物理的ブロックの掛け合わせであり、極めて高い再現性が期待できます。

\- \*\*過学習（Overfit）の懸念が最も高い仮説:\*\* `\[H-004]` です。「波高 >= 5cm」という条件は、対象となる4ヶ月間において特定の競艇場（例：冬場の江戸川や鳴門など）の特定の日数にサンプルが極端に偏る可能性が高く、全国的な汎用性よりも特定場の局所的な傾向を拾い上げてしまうリスクがあります。
