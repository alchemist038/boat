125 LINE PROJECT STATUS FOR NOTEBOOKLM

1. 目的
このファイルは、ボートレース3連単の探索枝「125ライン」に関する考察、作業履歴、現時点の結論を、NotebookLMやGeminiやGPTに渡しやすいように整理した要約である。

2. 基本発想
125ラインの発想は次の通り。

- 1号艇は弱そうに見える
  - 例: A1ではない、モーターが弱い、展示で2号艇/3号艇に見劣る
- ただし「1を完全に消す」より、「1頭のまま相手を絞る」方が市場歪みに合う
- 特に 5号艇/6号艇 のどちらかに格下がいて、1艇だけ事前に消せるなら、買い目が絞れるのではないか

この発想から、
「6号艇に格下が来て消せる、かつ1号艇が弱そうに見えるときの 1-2-5」
を中心に掘る枝が 125ライン である。

3. 探索期間
- 探索期間: 2023-03-11..2025-12-31
- 2026年は前方ホールドアウトとして残す

4. C2からの分岐
もともとはC2系統を掘っていたが、その後いったんC2を横に置き、
- day/night 仮説
- weak-looking lane1 + removable outer one
という別枝へ進んだ。

125ラインは、この「weak-looking lane1 + removable outer one」から生まれた。

5. 先に見えたこと
ベースラインでは、
- 1-ALL-ALL の方が 1飛び全体より ROI が高い
- つまり「1を丸ごと消す」方向は弱い

その一方で、
- 1-2-ALL は 1頭系の中で一番マシ
- 1-3/4/5/6-ALL が足を引っ張っている

このため、
「1を消す」より
「1頭を残したまま、相手の広がりすぎを削る」
方が自然だと判断した。

6. removable outer 仮説
最初に考えた removable outer は2種類。

- remove_b2_56
  - 5号艇または6号艇に B2 がいるなら、その弱い方を1艇消す
- remove_outer_worst_motor_456
  - 4号艇/5号艇/6号艇の中でモーター最下位を1艇消す

結果:
- remove_b2_56 は明確に効いた
- remove_outer_worst_motor_456 はほぼ弱かった

代表値:
- all + remove_b2_56
  - races = 33,314
  - 1-ALL-ALL ROI 63.17% -> 圧縮後 75.75%
  - uplift = +12.58pt
- lane1_motor_q25 + remove_b2_56
  - races = 12,090
  - 61.87% -> 73.98%

ここで重要なのは、
「外枠の弱さ」をモーター単体より、B2 という市場が見ている弱さで切った方が機能したこと。

7. F持ちは未使用
当初は
- 1号艇のF持ち
も使いたかったが、探索期間の DB では entries.f_count が実質すべて NULL だった。

そのため、この枝では F持ちは使えていない。
現時点で使えた 1号艇弱化条件は主に次の3本。

- lane1_motor_q25
  - 1号艇モーターが6艇中下位25%
- lane1_class_not_a1
  - 1号艇がA1ではない
- lane1_exgap_002
  - 1号艇の展示タイムが 2号艇または3号艇より 0.02秒以上見劣る

8. 1-2-remaining_outer まで絞った結果
remove_b2_56 を固定して総当たりしたところ、広い買い方の中で本当に伸びたのはほぼ
- 1-2-remaining_outer
だけだった。

ただし、
- 1〜2レース/日 くらいの実用密度に寄せると、全場平均ではまだ 100% を少し割った

例:
- lane1_class_not_a1 x night x 1-2-remaining_outer
  - races = 1,298
  - avg/day = 1.26
  - ROI = 95.35%
- all x night x 1-2-remaining_outer
  - races = 2,052
  - avg/day = 2.00
  - ROI = 93.36%

9. 10倍以下除外は逆効果
次に「10倍以下の低配当を除外したら良くなるか」を見たが、これは悪化した。

これは事後 oracle フィルタであり、実運用条件ではないが、参考値としても悪かった。

例:
- class_not_a1_and_exgap_002 x night x 1-2-remaining_outer
  - baseline ROI = 111.82%
  - 10倍以下除外 oracle ROI = 100.26%
- lane1_class_not_a1 x night x 1-2-remaining_outer
  - baseline ROI = 95.35%
  - oracle ROI = 82.21%

結論:
- この枝は「妙味だけ取る枝」ではない
- 低配当ヒットも含めて全体で支えるタイプ

10. rem5 / rem6 の発見
ここで本質だったのが、remaining_outer を 5号艇 と 6号艇 に分けることだった。

結果:
- rem5 は強い
- rem6 はかなり弱い

意味:
- rem5 = 5/6 のうち弱い方を消したあと、残る外枠が5号艇
- 実質、6号艇を消して 1-2-5 を狙う形
- rem6 = 実質 1-2-6 で、これは壊れていた

代表値:
- all x night x 1-2-remaining_outer
  - rem5: races 1,247, avg/day 1.214, ROI 106.05%
  - rem6: races 805, avg/day 0.784, ROI 73.71%
- lane1_class_not_a1 x night
  - rem5 ROI 114.68%
  - rem6 ROI 68.47%
- lane1_exgap_002 x night
  - rem5 ROI 136.15%
  - rem6 ROI 43.52%
- class_not_a1_and_exgap_002 x night
  - rem5 ROI 164.21%
  - rem6 ROI 43.08%

この時点で、この枝はもはや
- 1-2-remaining_outer
ではなく、
- 実質 1-2-5
と見るべきだと判断した。

11. 125ラインの定義
ここから、rem5 枝を 125ライン と呼ぶ。

意味は単純で、
- 6号艇に格下(B2)が来ていて消せる
- 5号艇が残る
- 1号艇は弱そうに見える
- そのときの 1-2-5 を狙う

つまり一文でいうと、
「6に格下がいて消せる、かつ1が弱そうに見えるときの 1-2-5」
である。

17. 現時点の結論
125ラインは「1を消す」戦略ではない。
本質は、
- 1は弱そうに見える
- 6は格下で消せる
- そのとき市場が 1-2-5 を詰め切れていない
という歪みを狙う枝である。

この枝は有望だが、まだ正規ロジックではない。

- 有望な探索枝
