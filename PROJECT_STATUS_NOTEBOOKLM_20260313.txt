======================================================================
1. プロジェクト全体の目的
======================================================================

このプロジェクトの主目的は、BOAT RACE公式ベースのデータをローカルに蓄積し、
後から統計分析・バックテスト・期待値検証を再現可能な形で行えるようにすること。

優先順位は一貫して以下:
- 公式データ中心の収集
- raw と整形済みデータの分離
- 再取得可能な保存構造
- DuckDB による分析しやすい正規化
- 戦略BTの結果を履歴として残すこと

重要:
- 戦略の勝ち負けより先に、データ品質と再現性を優先する
- 同じロジックを条件だけ変えて何度も再実行しないよう、履歴台帳を残す

6. 発見されたデータ不具合
======================================================================

問題箇所:
- D:\boat\src\boat_race_data\mbrace.py
- parse_mbrace_k_results 周辺

症状:
- 5着 / 6着行で course_entry が極端に欠損
- finish順位に course_entry present rate が依存
- その結果、Aセットが成立しやすいレースだけ observed 側に偏って見えた

- 1着: ほぼ100%
- 2着: ほぼ100%

- 5着: 約26%
- 6着: 約25%

原因:
- raw の mbrace_k TXT では、5着/6着行で race_time が空でも
  末尾に ". ." のプレースホルダ列が残るケースがある
- 旧 parser は「race_time がない = 列もない」と誤解し、
  course_entry / exhibition_time / start_token の列位置を左にずらして解釈していた

結果:
- 5着/6着の course_entry 復元不良
- 観測あり群が人工的に有利な母集団になる
- Aラインの見かけ成績が過大評価される

15. C2ロジック作業履歴
======================================================================

1. 出発点
- もともとの問題意識は「1号艇が飛んだときは高配当だが、本当に期待値があるのか」
- 全体ベースラインでは
  - 1-ALL-ALL ROI は約70.9%
  - 1飛び(1以外-ALL-ALL) ROI は約58.0%
- つまり「1飛び全体」は弱く、局所的な市場の歪みを探す方向へ移行した

2. A/B/C 仮説のうち C 系が生き残った
- A: 江戸川 x 気象
- B: 1-2-ALL x 展示タイム
- C: 女子戦 x 1号艇の展示ST遅れ
- このうち実際に100%超えまで届いたのは C 系統だった

3. C0 / C1 / C2 の定義
- C0: 女子戦全体で 2-ALL-ALL / 3-ALL-ALL を買う
- C1: 女子戦かつ st1 >= 0.20
- C2: 女子戦かつ st1 - min(st2..st6) >= 0.10
- ここで ST は start_exhibition_st、数値が大きいほどスタート展示で遅れている

4. オリジナル C2 の主結果
- 対象期間は本線全期間 2023-03-11..2026-03-10
- C2 オリジナル(女子戦 x ST遅れ>=0.10) で
  - races 1,891
  - 2-ALL-ALL ROI 110.55%
  - 3-ALL-ALL ROI 108.35%
  - 2/3-ALL-ALL 合算 ROI 109.45%
- ここで初めて「市場が1号艇を過信している可能性」が具体的な数字で見えた

5. DD / 月次の確認
- C2 合算の累積収支を確認したところ
  - 総投資 7,564,000円
  - 総払戻 8,279,060円
  - 総利益 715,060円
  - 最大DD 265,880円
- 月次はかなり荒く、20レース以上ある月は36か月中36
  - そのうち月次ROI100%以上は19か月
- 「一発の万舟だけ」ではないが、「安定右肩上がり」と言い切るには荒い、という判定になった

6. C2 の解像度アップ
- C2 母集団を 場別 / 風速帯 / 展示タイム差 で切った
- 重要だったのは次の3点
  - 風速0-2m は ROI 125.28% と強い
  - ST遅れ 0.16以上 は ROI 147.55% と最も強い
  - ただし「2号艇/3号艇の展示優位」が本体ではなく、むしろ ex1 が同等以上の側が強かった
- つまり本質は「1号艇がSTでは危ないのに、市場はなお1を見てしまう」局面だった

7. 厳格版 refined C2
- 条件を
  - 女子戦
  - st1 - min(st2..st6) >= 0.10
  - wind <= 2
  - ex1 <= ex2 かつ ex1 <= ex3
 まで絞った
- 結果
  - races 387
  - ROI 148.39%
  - PF 1.7186
  - 最大DD 147,130円
- ただし件数が細すぎて、月次サンプルが薄く本命ロジックには早いと判断した

8. グリッド探索と暫定本線
- ST閾値 / 風速上限 / 展示タイム許容差(ex slack) を総当たりした
- 倍サンプル帯で最も良かった候補は

  - ex1 <= ex3 + 0.02
  - wind 制限なし
  - 買い目 2-ALL-ALL + 3-ALL-ALL
