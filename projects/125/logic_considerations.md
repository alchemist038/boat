# 125 Logic Considerations

## 検討項目

### 展示スタートの並び正常

125 ロジックの追加検討項目として、`展示スタートの並び正常` を入れておく。

意図:

- 1号艇優勢に見えても、展示スタートの並びが崩れているだけのケースを弾きたい
- 1-2-5 系の形が、展示の一時的な乱れではなく、素直な隊形から出ているかを確認したい
- `lane1 best` だけでなく、`隊形として無理がないか` を見る補助条件として使いたい

現時点の扱い:

- すぐに本採用する条件ではなく、まずは `検討項目` として保持
- いきなり hard filter にせず、最初は
  - 条件メモ
  - 失敗レースの振り返り
  - BT / FW の差分確認
  から入る

考えたい観点:

- 1号艇の展示 ST が十分に前にいるか
- 2号艇、3号艇との並びが自然か
- 外の 4-5-6 に極端な先行がいて、1-2-5 形を壊していないか
- `lane1 best` でも、展示 ST の並びが乱れているだけの見かけ優位ではないか
- スタート隊形が正常なときだけ成績が安定するか

確認したいこと:

- 住之江単場 1-2-5 で効くか
- 鳴門 / 芦屋 / 江戸川を含む broad four stadium でも効くか
- `filtered_out` にすべき誤反応を減らせるか
- 逆に、有効レースを削りすぎないか

実装方針の候補:

- まずは `補助フラグ` として出す
- 次に BT / FW で差分を見る
- 効くなら
  - hard filter
  - score 加点 / 減点
  - warning flag
  のどれにするかを決める

メモ:

- 今回の論点は `展示タイム単体` ではなく、`展示スタートの並びが正常か` を見ること
- したがって、単純な ST 最速条件ではなく、`並び` や `隊形` の見方として整理する
