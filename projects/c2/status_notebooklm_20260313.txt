C2 PROJECT STATUS FOR NOTEBOOKLM

1. 目的
このファイルは、ボートレース3連単の市場歪み仮説「C2」に関する作業履歴だけを、NotebookLMやGeminiやGPTに渡しやすいように整理した要約である。

2. 出発点
出発点は「1号艇が飛んだときは高配当だが、本当に期待値があるのか」という問題意識だった。
全体ベースラインでは、
- 1-ALL-ALL ROI 約70.9%
- 1飛び(1以外-ALL-ALL) ROI 約58.0%
となり、「1飛び全体」を広く買う作戦は弱いと分かった。
そこで、1号艇が危ないのに市場がなお1号艇を買いすぎる局所条件を探す方向へ移った。

3. A/B/C仮説
- A: 江戸川 x 気象
- B: 1-2-ALL x 展示タイム
- C: 女子戦 x 1号艇の展示ST遅れ

この中で、実際にROI 100%超まで届いたのはC系統だった。

4. C0 / C1 / C2 の定義
- C0: 女子戦全体で 2-ALL-ALL / 3-ALL-ALL を買う
- C1: 女子戦かつ st1 >= 0.20
- C2: 女子戦かつ st1 - min(st2..st6) >= 0.10

用語:
- ST = start_exhibition_st
- 数値が大きいほどスタート展示で遅れている

5. オリジナルC2の結果
対象期間:
- 2023-03-11..2026-03-10

結果:
- C2母集団 races = 1,891
- 2-ALL-ALL ROI = 110.55%
- 3-ALL-ALL ROI = 108.35%
- 2/3-ALL-ALL 合算 ROI = 109.45%

ここで、「1号艇は危ないのに市場はまだ1を見すぎている」可能性が初めて数字で見えた。

6. DDと月次
C2合算の累積収支では、
- 総投資額 = 7,564,000円
- 総払戻額 = 8,279,060円
- 総利益 = 715,060円
- 最大DD = 265,880円

月次では、
- 20レース以上ある月 = 36
- そのうち月次ROI 100%以上 = 19

評価:
- 一発の万舟だけで成立したロジックではない
- ただし月次のブレは大きく、安定右肩上がりとはまだ言いにくい

7. C2の解像度アップ
C2母集団をさらに分解した結果、重要だったのは次の点だった。

- 風速0-2m は ROI 125.28% と強い
- ST遅れ 0.16以上 は ROI 147.55% と最も強い
- 2号艇や3号艇の展示タイム優位が本体ではなかった
- むしろ「1号艇はSTでは危ないが、EXではそこまで悪く見えない」局面で歪みが出ていた

つまりC2の本質は、
「1号艇がスタート展示でかなり危ないのに、市場はなお1号艇を見てしまう」
というギャップだった。

8. 厳格版 refined C2
条件:
- 女子戦

- wind <= 2
- ex1 <= ex2
- ex1 <= ex3

結果:

評価:
- 数字はかなり良い
- ただし件数が細すぎて、本命ロジックにするにはまだ早い

9. グリッド探索
次に、ST閾値、風速上限、展示タイム許容差を総当たりした。

その結果、倍くらいのサンプルを保ちながら強かった候補はこれだった。

条件:
- 女子戦 proxy
- st1 - min(st2..st6) >= 0.12
- ex1 <= ex2 + 0.02
- ex1 <= ex3 + 0.02
- wind 制限なし
- 買い目 = 2-ALL-ALL + 3-ALL-ALL

結果:
- races = 763
- ROI = 145.01%
- PF = 1.7160
- 最大DD = 149,590円
- 月20件以上ある月 = 18
- そのうち月次ROI 100%以上 = 10

この候補を Strategy_C2_Provisional_v1 として固定した。

10. 現時点の位置づけ
C2は崩れたロジックではない。
むしろ市場歪みの検出としては今も有望である。

ただし、まだ次の課題がある。
- 女子戦判定が proxy ベース
- 月次のブレが大きい
- 2026年で前方フォワード確認が未実施

したがって、現時点の分類は
- 正規ロジックではない
- 準正規ロジック
である。

11. 現在の扱い
Strategy_C2_Provisional_v1 は「保持すべき有望枝」だが、「本運用確定ロジック」ではない。

その後は、C2を候補として保持したまま、別枝の探索に進んでいる。
現在の別枝は主に次の2本。
- day/night 仮説
- weak-looking lane1 + removable outer one

この新枝では、2026を前方ホールドアウトとして残し、探索期間を 2023-03-11..2025-12-31 に固定している。

12. C2に関する現時点の結論
- C2は死んでいない
- 100%超えは確認済み
- ただし全場共通の即運用ロジックにはまだ早い
- 2026フォワードで再確認する価値が高い
