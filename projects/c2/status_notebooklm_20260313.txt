C2 PROJECT STATUS FOR NOTEBOOKLM

1. 目的
このファイルは、ボートレース3連単の市場歪み仮説「C2」に関する作業履歴だけを、NotebookLMやGeminiやGPTに渡しやすいように整理した要約である。

2. 出発点
出発点は「1号艇が飛んだときは高配当だが、本当に期待値があるのか」という問題意識だった。
全体ベースラインでは、
- 1-ALL-ALL ROI 約70.9%
- 1飛び(1以外-ALL-ALL) ROI 約58.0%
となり、「1飛び全体」を広く買う作戦は弱いと分かった。
そこで、1号艇が危ないのに市場がなお1号艇を買いすぎる局所条件を探す方向へ移った。

3. A/B/C仮説
- A: 江戸川 x 気象
- B: 1-2-ALL x 展示タイム
- C: 女子戦 x 1号艇の展示ST遅れ

この中で、実際にROI 100%超まで届いたのはC系統だった。

4. C0 / C1 / C2 の定義
- C0: 女子戦全体で 2-ALL-ALL / 3-ALL-ALL を買う
- C1: 女子戦かつ st1 >= 0.20
- C2: 女子戦かつ st1 - min(st2..st6) >= 0.10

用語:
- ST = start_exhibition_st
- 数値が大きいほどスタート展示で遅れている

5. オリジナルC2の結果
対象期間:
- 2023-03-11..2026-03-10

結果:
- C2母集団 races = 1,891
- 2-ALL-ALL ROI = 110.55%
- 3-ALL-ALL ROI = 108.35%
- 2/3-ALL-ALL 合算 ROI = 109.45%

ここで、「1号艇は危ないのに市場はまだ1を見すぎている」可能性が初めて数字で見えた。

7. C2の解像度アップ
C2母集団をさらに分解した結果、重要だったのは次の点だった。

- 風速0-2m は ROI 125.28% と強い
- ST遅れ 0.16以上 は ROI 147.55% と最も強い
- 2号艇や3号艇の展示タイム優位が本体ではなかった
- むしろ「1号艇はSTでは危ないが、EXではそこまで悪く見えない」局面で歪みが出ていた

つまりC2の本質は、
「1号艇がスタート展示でかなり危ないのに、市場はなお1号艇を見てしまう」
というギャップだった。
